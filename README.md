## 🛠 Getting Started

### 1️⃣ Prerequisites

- Node.js (v16+)
- npm or yarn
- Git

### 2️⃣ Installation

```sh
# Clone the repository
git clone https://github.com/jagdishsaini90/Globetrotter-client.git
cd trivia-city-game

# Install dependencies
npm install

# Start the development server
npm run dev
```
